# Evidence quality

Provider edge evidence is scored with explicit components: directedness, signed effect,
publication support, independent original database support, curated status, predicted-only penalty,
functional-only penalty, and uncertainty class.

Evidence levels are `high`, `moderate`, `low`, `contextual`, `conflicting`, and `insufficient`.
Unsigned functional interactions are contextual and are not treated as causal direction evidence.
Conflicting signed support remains explicit and is carried into consensus warnings.
