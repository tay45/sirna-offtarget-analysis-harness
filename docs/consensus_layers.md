# Consensus layers

Provider evidence is lineage-deduplicated before consensus. Duplicates sharing source, target,
relation type, sign, original source, and primary reference are merged with combined references and
warnings.

Consensus edges preserve provider sources, evidence IDs, lineage groups, references, signed support
counts, unsigned support counts, conflict counts, database versions, retrieval snapshots, and causal
eligibility. Reactome FI functional interactions remain unsigned/contextual.
