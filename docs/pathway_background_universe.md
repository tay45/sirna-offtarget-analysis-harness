# Pathway Background Universe

The default background is `tested_detectable_annotatable_genes`: genes must be
expression-tested, detectable, identifier-valid, and present in selected provider
annotation membership when membership is available.

The stage persists `background_universe.tsv`, `background_exclusions.tsv`, and
`background_manifest.json` under the committed pathway enrichment attempt.
Whole-genome background is not silently used.
