# Intended Target Site Validation

Transcript presence alone does not validate the intended target. When an intended-target
transcript is supplied, transcript targetability requires an actual compatible site in that
transcript sequence.

The validation checks transcript availability, site evidence class, total mismatches, seed
mismatches, and central mismatches. Expression decrease is not used.

Gene-only intended targets are preserved as uncertainty with a warning because there is no
single transcript sequence to validate.
