# Transcript Targetability Ratio Verification

The verifier recomputes transcript sets, N, M, M/N, seed-only exclusion,
unavailable evidence handling, failed-gene behavior, and equal-prior consistency
from canonical artifacts.

It rejects site-count inflation, seed-only inclusion under the default policy,
M greater than N, definitive ratios with unresolved evidence, and zero-denominator
ratios.
