# Biological Problem

siRNA treatment can reduce expression through intended or unintended transcript
targeting. It can also change expression indirectly through pathway response,
regulatory feedback, stress signaling, or other downstream effects.

Interpreting these signals is harder when genes have multiple transcripts. A
guide may target one transcript but not another, and short-read data may not
resolve transcript abundance. This repository keeps expression, transcript
targetability, isoform uncertainty, and pathway evidence separate so later
integration can remain auditable.
