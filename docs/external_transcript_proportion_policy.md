# External Transcript Proportion Policy

Validated external transcript proportions are optional evidence.

Required evidence includes gene ID, transcript ID, canonical identifiers, proportion, scope, method, software, software version, annotation release, organism, assembly, and source checksum.

Validation rejects nonnumeric, negative, duplicate, wrong-gene, organism-mismatched, assembly-mismatched, release-mismatched, ineligible-transcript, and materially non-summing records.

Default production behavior does not silently renormalize materially inconsistent values.
