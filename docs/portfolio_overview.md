# Portfolio Overview

This repository presents a reproducible siRNA off-target analysis harness as a
bioinformatics portfolio project. The current release validates evidence
preservation, transcript targetability, isoform uncertainty, pathway evidence
architecture, typed contracts, provenance, checksums, resume behavior, and
independent verification.

It is intentionally not a completed final classifier. The current release
prepares trustworthy evidence layers for later direct-versus-secondary
integration.
