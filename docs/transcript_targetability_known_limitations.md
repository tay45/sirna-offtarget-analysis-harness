# Transcript Targetability Known Limitations

Transcript targetability records sequence evidence only.

It does not compute formal transcript count N, targetable transcript count M, M/N,
intended-target knockdown calibration, expected direct effect, residuals, secondary
effects, source eligibility, or final classification.

Failed genes are preserved as transcript targetability failure evidence but are
not converted into formal M exclusions until the future M definition is written.
Passenger-strand search remains unsupported.
