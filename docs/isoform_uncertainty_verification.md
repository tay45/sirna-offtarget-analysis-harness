# Isoform Uncertainty Verification

Verification rejects:

- equal prior labeled as observed abundance,
- transcript-specific fold change or p-value fields,
- negative, nonfinite, or greater-than-one weights,
- weights for zero eligible transcripts,
- unequal equal-prior weights,
- invalid external proportions,
- organism, assembly, or annotation release mismatch,
- unverified production annotations,
- missing output checksums on completed runs,
- targetability, M, N, M/N, intended-target calibration, direct-effect, secondary-effect, or classification logic in this stage.
