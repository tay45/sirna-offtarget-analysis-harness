# Transcript Targetability Ratio Contracts

The ratio stage writes `TranscriptTargetabilityRatioResultV1`,
`TranscriptTargetabilityRatioRunRecordV1`,
`GeneTranscriptTargetabilityRatioRecordV1`,
`TranscriptMContributionRecordV1`, and
`TranscriptTargetabilityRatioVerificationRecordV1`.

Unavailable values are null with typed status fields, never sentinel values.
