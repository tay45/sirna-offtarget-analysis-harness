# Next Stage N/M Ratio Boundary

The next stage may define formal N, M, and M/N. It must consume transcript targetability
evidence explicitly and define how unavailable transcript sequences are handled.

No N, M, or M/N value is produced by transcript targetability.

The next pass should consume the completed targetability evidence, unavailable
sequence evidence, and gene-failure records explicitly. It must define whether
and how failed genes affect future M eligibility before calculating any ratio.
