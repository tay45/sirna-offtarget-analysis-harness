# Isoform Uncertainty Known Limitations

Implemented:

- explicit annotation snapshot metadata,
- deterministic transcript eligibility,
- equal-transcript prior,
- optional validated precomputed transcript proportions,
- typed records and artifact writer.

Not implemented:

- transcript quantification,
- abundance-derived proportion conversion,
- workflow runner execution of the new stage,
- siRNA transcript matching,
- targetable transcript count M,
- formal off-target N,
- M/N,
- direct, secondary, or classification logic.
